
# Project Blueprint: Interactive 3D House Explorer

## Overview

This project is a web-based application that allows users to explore a 3D model of a house. Users can navigate through different rooms and interact with various objects to learn about their chemical components. The application is built using React and Three.js.

## Implemented Features

*   **Project Setup:**
    *   Initialized a React project using Vite.
    *   Installed core 3D rendering libraries: `three`, `@react-three/fiber`, and `@react-three/drei`.
    *   Established a `public/models` directory to store 3D model assets.

## Current Plan

*   **Phase 1: Basic Scene Setup**
    1.  Install Three.js and related dependencies. (Done)
    2.  Create a dedicated folder for 3D models. (Done)
    3.  Clear the default boilerplate from the main application component.
    4.  Set up a basic 3D scene using `@react-three/fiber`.
    5.  Add a simple geometric shape (e.g., a spinning cube) to verify the 3D canvas is working correctly.
    6.  Implement basic camera controls to allow users to look around the scene.
